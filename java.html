String Example Array Example ArrayList Example
Definition String: a sequence of characters used to represent text. Array: a fixed-size data structure that holds a collection of elements of the same type contiguously in memory. ArrayList: a resizable data structure defined as a Class that holds a collection of elements and allows for dynamic addition and removal of elements.
What is this type used for? String: Used for storing and manipulating sequences of characters, such as text, names, or messages. It provides various methods for operations like searching, replacing, and formatting text. Array: Used for storing a fixed-size collection of elements of the same type, allowing for efficient access and modification of elements using an index. Arrays are ideal when the number of elements is known and fixed at compile time. ArrayList: Used for storing a dynamic collection of elements that can grow or shrink in size. It provides flexibility with operations such as adding, removing, and accessing elements, making it useful when the number of elements is not known in advance or changes frequently.

Why would you use a traditional for loop over an enhanced for-each loop? Describe a scenario for each. While the enhanced for-each loop is often more concise and readable, there are situations where a traditional for loop might be preferable. Traditional For Loop: If you want to modify the elements of the collection while iterating, you need to use a traditional for loop to have access to the index. Enhanced For-Each Loop: If you're only reading the values of the elements and not modifying them, the enhanced for-each loop is a good choice.
